11:T3164,<p>「来月の支払い、足りるかな…？」</p>
<p>この不安を解消するのが<strong>資金繰り表</strong>です。個人事業主であっても、お金の流れを「見える化」するだけで、資金ショートのリスクを事前に察知し、適切な対策を打てるようになります。</p>
<p>この記事では、資金繰り表の基本、Excelで簡単に作れる5ステップ、そして資金繰りを改善するポイントを解説します。</p>
<hr>
<h2>資金繰り表とは？</h2>
<p>資金繰り表とは、<strong>一定期間の現金の入出金を予測・管理するための表</strong>です。損益計算書が「利益」を管理するのに対し、資金繰り表は「現金（キャッシュ）」を管理します。</p>
<h3>損益計算書との違い</h3>
<p>| 項目 | 損益計算書 | 資金繰り表 |
|------|---------|----------|
| 管理対象 | 利益（売上−経費） | 現金（入金−出金） |
| 計上タイミング | 発生主義（取引発生時） | 現金主義（入出金時） |
| 減価償却費 | 経費に含む | 含まない（現金流出なし） |
| 掛取引 | 売上計上済みでも未入金あり | 実際の入金タイミングで記録 |
| 目的 | 経営成績の把握 | 資金ショートの防止 |</p>
<p><strong>重要:</strong> 利益が出ていても現金が足りなくなる「黒字倒産」は、資金繰り表がない個人事業主に起こりがちです。売上が増えるほど仕入れ・外注費も増え、入金よりも出金が先行する構造になりやすいからです。</p>
<hr>
<h2>資金繰り表を作る5ステップ</h2>
<h3>ステップ1: テンプレートを用意する</h3>
<p>ExcelまたはGoogleスプレッドシートで以下の項目を横軸（月）と縦軸（項目）で作成します。</p>
<p><strong>基本レイアウト:</strong></p>
<pre><code>              | 4月  | 5月  | 6月  | 7月  | 8月  | 9月  |
─────────────┼─────┼─────┼─────┼─────┼─────┼─────┤
【前月繰越残高】 |      |      |      |      |      |      |
─────────────┼─────┼─────┼─────┼─────┼─────┼─────┤
【収入の部】    |      |      |      |      |      |      |
 現金売上      |      |      |      |      |      |      |
 売掛金回収    |      |      |      |      |      |      |
 その他収入    |      |      |      |      |      |      |
 収入合計      |      |      |      |      |      |      |
─────────────┼─────┼─────┼─────┼─────┼─────┼─────┤
【支出の部】    |      |      |      |      |      |      |
 仕入れ       |      |      |      |      |      |      |
 外注費       |      |      |      |      |      |      |
 人件費       |      |      |      |      |      |      |
 家賃        |      |      |      |      |      |      |
 光熱費・通信費 |      |      |      |      |      |      |
 借入金返済    |      |      |      |      |      |      |
 税金・保険料   |      |      |      |      |      |      |
 その他支出    |      |      |      |      |      |      |
 支出合計      |      |      |      |      |      |      |
─────────────┼─────┼─────┼─────┼─────┼─────┼─────┤
【収支差額】    |      |      |      |      |      |      |
【翌月繰越残高】 |      |      |      |      |      |      |
</code></pre>
<h3>ステップ2: 前月繰越残高を記入する</h3>
<p>事業用口座の残高＋手元現金の合計を「前月繰越残高」に記入します。<strong>必ず事業用口座の実際の残高と一致させてください。</strong></p>
<h3>ステップ3: 収入を予測して記入する</h3>
<p>各月の収入を「入金されるタイミング」で記入します。</p>
<p><strong>注意点:</strong></p>
<ul>
<li>売掛金は「売上計上月」ではなく「入金予定月」に記入する</li>
<li>新規顧客からの入金は確実性を考慮して控えめに見積もる</li>
<li>季節変動がある業種は過去の実績を参考にする</li>
</ul>
<h3>ステップ4: 支出を予測して記入する</h3>
<p>各月の支出を「支払うタイミング」で記入します。</p>
<p><strong>見落としやすい支出:</strong></p>
<ul>
<li>年1回の支払い（確定申告の所得税・住民税・個人事業税）</li>
<li>賞与・ボーナス月の人件費</li>
<li>設備の更新・修繕費</li>
<li>保険料の年払い</li>
<li>消費税の納付（前々年の売上が1,000万円超の場合）</li>
</ul>
<h3>ステップ5: 収支差額と翌月繰越残高を計算する</h3>
<pre><code>収支差額 ＝ 収入合計 − 支出合計
翌月繰越残高 ＝ 前月繰越残高 ＋ 収支差額
</code></pre>
<p><strong>翌月繰越残高がマイナスになる月があれば要注意です。</strong> そのままでは資金ショートするため、事前に対策を講じる必要があります。</p>
<hr>
<h2>資金繰り表の記入例（IT系フリーランスの場合）</h2>
<p>| 項目 | 4月 | 5月 | 6月 |
|------|-----|-----|-----|
| <strong>前月繰越残高</strong> | 120万円 | 95万円 | 70万円 |
| <strong>【収入の部】</strong> | | | |
| 売掛金回収（A社） | 50万円 | 50万円 | 50万円 |
| 売掛金回収（B社） | 30万円 | 30万円 | 30万円 |
| その他収入 | 0万円 | 5万円 | 0万円 |
| <strong>収入合計</strong> | <strong>80万円</strong> | <strong>85万円</strong> | <strong>80万円</strong> |
| <strong>【支出の部】</strong> | | | |
| 外注費 | 30万円 | 30万円 | 30万円 |
| 家賃 | 8万円 | 8万円 | 8万円 |
| 通信費・ツール | 3万円 | 3万円 | 3万円 |
| 生活費（事業主貸） | 40万円 | 40万円 | 40万円 |
| 税金・保険料 | 10万円 | 10万円 | 10万円 |
| その他 | 14万円 | 19万円 | 5万円 |
| <strong>支出合計</strong> | <strong>105万円</strong> | <strong>110万円</strong> | <strong>96万円</strong> |
| <strong>収支差額</strong> | <strong>-25万円</strong> | <strong>-25万円</strong> | <strong>-16万円</strong> |
| <strong>翌月繰越残高</strong> | <strong>95万円</strong> | <strong>70万円</strong> | <strong>54万円</strong> |</p>
<p>この例では、毎月25万円前後の赤字が続いており、<strong>7〜8月頃に資金ショートするリスク</strong>があることがわかります。早い段階で新規案件の獲得や、支出の見直しが必要です。</p>
<hr>
<h2>資金繰り表を活用した改善ポイント5つ</h2>
<h3>1. 入金サイトを短くする</h3>
<p>売掛金の回収を早めることが、資金繰り改善の最も効果的な方法です。</p>
<ul>
<li>取引先に支払いサイトの短縮を交渉する（月末締め翌々月払い→翌月払い）</li>
<li>早期支払い割引を提案する</li>
<li>ファクタリングで売掛金を即日現金化する</li>
</ul>
<h3>2. 支払いサイトを長くする</h3>
<p>仕入れ先や外注先への支払いを、可能な範囲で後ろ倒しにします。ただし、<strong>取引先との関係を悪化させない範囲で</strong>行うことが重要です。</p>
<h3>3. 固定費を削減する</h3>
<p>毎月の固定費を見直すだけで、資金繰りは大幅に改善します。</p>
<p>| 見直し項目 | 具体的なアクション | 月間削減額の目安 |
|-----------|----------------|---------------|
| 家賃 | コワーキングスペース活用、移転 | 3〜10万円 |
| 通信費 | プランの見直し、格安SIM | 0.5〜2万円 |
| サブスクリプション | 不要サービスの解約 | 1〜5万円 |
| 保険料 | 補償内容の見直し | 0.5〜2万円 |</p>
<h3>4. 繁忙期・閑散期に備える</h3>
<p>季節変動がある業種では、<strong>繁忙期の利益を閑散期の資金に回す</strong>計画を立てましょう。資金繰り表で年間の波を可視化することで、「いつ、いくら準備しておくべきか」が明確になります。</p>
<h3>5. 資金ショートの「3か月前」に動く</h3>
<p>資金繰り表でマイナスが見えたら、<strong>最低3か月前に対策を始めましょう。</strong> 融資の申込みから入金まで2〜4週間はかかるため、直前では間に合いません。</p>
<hr>
<h2>資金繰り表を作る際のよくある間違い</h2>
<h3>間違い1: 売上と入金を混同する</h3>
<p>「今月100万円の仕事を受注した＝今月100万円入る」ではありません。請求書を発行してから実際に入金されるまでのタイムラグ（支払いサイト）を正確に反映させましょう。</p>
<h3>間違い2: 楽観的な予測をする</h3>
<p>新規の見込み客を「たぶん受注できるだろう」と収入に入れてしまうと、計画と実績が乖離します。<strong>確定した案件のみを収入に計上し、見込み案件は別枠で管理する</strong>のが安全です。</p>
<h3>間違い3: 税金の支払いを忘れる</h3>
<p>所得税、住民税、消費税、個人事業税など、年に1〜4回まとめて支払う税金は見落としやすい項目です。前年の実績を参考に、必ず資金繰り表に組み込みましょう。</p>
<hr>
<h2>資金繰りが厳しくなったときの対処法</h2>
<p>資金繰り表でマイナスが見えたときの対処法を、優先度順に紹介します。</p>
<p>| 優先度 | 対処法 | 特徴 |
|--------|-------|------|
| 1 | 入金サイトの短縮交渉 | コストゼロ、関係次第 |
| 2 | 固定費の削減 | 即効性あり |
| 3 | 融資の申し込み | 低金利だが時間がかかる |
| 4 | ファクタリングの利用 | 即日可能、手数料あり |
| 5 | ビジネスローン | 即日可能、高金利 |</p>
<p>フリーランスの資金繰り改善については<a href="/articles/article-freelance-shikin/">フリーランスの資金繰りガイド</a>で詳しく解説しています。</p>
<hr>
<h2>まとめ</h2>
<p>資金繰り表は、個人事業主が「お金の不安」から解放されるための最も基本的なツールです。</p>
<p><strong>作成の5ステップ:</strong></p>
<ol>
<li>テンプレートを用意する</li>
<li>前月繰越残高を記入する</li>
<li>収入を「入金タイミング」で予測する</li>
<li>支出を「支払いタイミング」で予測する</li>
<li>収支差額と翌月繰越残高を計算する</li>
</ol>
<p><strong>改善のポイント:</strong></p>
<ul>
<li>入金サイトを短く、支払いサイトを長く</li>
<li>固定費を定期的に見直す</li>
<li>マイナスが見えたら3か月前に行動する</li>
</ul>
<p>まだ資金繰り表を作っていない方は、今日からExcelで簡単なものを作ってみてください。月に1回更新するだけで、資金ショートのリスクを大幅に減らせます。資金調達の全体像については<a href="/articles/article-kojin-shikin-chotatsu/">個人事業主の資金調達方法10選</a>もあわせてご覧ください。</p>

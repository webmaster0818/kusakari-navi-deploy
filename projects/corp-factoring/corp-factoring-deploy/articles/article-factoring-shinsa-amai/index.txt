11:T2f56,<p>「銀行融資の審査に落ちた」「税金を滞納している」「赤字決算が続いている」——そんな状況でも、ファクタリングなら審査に通る可能性があります。</p>
<p>ファクタリングは銀行融資とは根本的に審査基準が異なります。この記事では、ファクタリング審査の仕組みから「審査が甘い」と言われる理由、通りやすい条件、そして審査に落ちる原因まで詳しく解説します。</p>
<hr>
<h2>ファクタリングの審査基準は「売掛先」が中心</h2>
<h3>なぜ審査が甘いと言われるのか</h3>
<p>銀行融資の審査では、申込企業の<strong>財務状況・信用情報・担保</strong>が重視されます。一方、ファクタリングの審査で最も重要なのは「<strong>売掛先の信用力</strong>」です。</p>
<p>| 審査項目 | 銀行融資 | ファクタリング |
|---------|---------|-------------|
| 申込者の財務状況 | ◎ 最重要 | △ 参考程度 |
| 申込者の信用情報 | ◎ 最重要 | △ 参考程度 |
| 担保・保証人 | ◎ 必要な場合多い | × 不要 |
| <strong>売掛先の信用力</strong> | × 関係なし | ◎ <strong>最重要</strong> |
| 売掛金の実在性 | × 関係なし | ◎ 重要 |</p>
<p><strong>ファクタリングは「売掛先が確実に支払えるか」を審査する仕組みのため、申込企業の業績が悪くても審査に通りやすい</strong>のです。</p>
<h3>ファクタリング審査で見られる主な項目</h3>
<p><strong>売掛先に関する確認事項：</strong></p>
<ul>
<li>売掛先の業歴・信用情報</li>
<li>支払い遅延の有無</li>
<li>業界の安定性</li>
<li>上場・非上場</li>
</ul>
<p><strong>売掛金に関する確認事項：</strong></p>
<ul>
<li>売掛金の実在性（架空でないか）</li>
<li>二重譲渡の有無（同じ売掛金を複数社に売却していないか）</li>
<li>支払期日</li>
<li>金額の妥当性</li>
</ul>
<p><strong>申込者に関する確認事項（参考程度）：</strong></p>
<ul>
<li>法人格の確認</li>
<li>代表者の本人確認</li>
<li>通帳の入出金状況</li>
</ul>
<hr>
<h2>審査が通りやすい条件</h2>
<h3>条件1：売掛先の信用力が高い</h3>
<p>最も審査に有利な条件は、売掛先が信用力の高い企業であることです。</p>
<p><strong>審査に通りやすい売掛先の例：</strong></p>
<ul>
<li>上場企業（東証プライム・スタンダード等）</li>
<li>大手企業（従業員1,000人以上）</li>
<li>国・地方公共団体（官公庁・市区町村）</li>
<li>独立行政法人・国立大学</li>
<li>大手医療機関（公立病院等）</li>
</ul>
<p><strong>これらの売掛先への請求書があれば、申込企業が赤字決算であっても審査通過の可能性が高くなります。</strong></p>
<h3>条件2：売掛金の支払期日が近い</h3>
<p>支払期日まで30〜60日程度の売掛金は審査が通りやすい傾向があります。ファクタリング会社にとって、回収までの期間が短いほどリスクが低くなります。</p>
<h3>条件3：通帳の入出金が安定している</h3>
<p>直近6ヶ月の通帳に大きな異常がない（急激な残高減少・頻繁な不審な入出金がない）ことが審査を通りやすくします。</p>
<h3>条件4：申込者が継続した事業実態を持つ</h3>
<ul>
<li>業歴1年以上（開業直後は対応できない会社もある）</li>
<li>定期的な売上がある（通帳で確認できる）</li>
<li>固定の取引先がある</li>
</ul>
<hr>
<h2>審査が通りにくい（落ちやすい）条件</h2>
<h3>原因1：売掛先の信用力が低い</h3>
<ul>
<li>設立間もない会社（業歴1年未満）</li>
<li>支払い遅延の前例がある会社</li>
<li>業績が悪化している会社</li>
<li>個人（法人格のない個人）</li>
<li>海外企業（外国企業への請求は対応不可の場合が多い）</li>
</ul>
<h3>原因2：売掛金に問題がある</h3>
<p><strong>架空の売掛金・二重譲渡</strong>は不正行為です。発覚した場合は契約解除だけでなく、法的責任を問われる場合もあります。</p>
<ul>
<li>架空の請求書（実際のサービス・商品提供が伴わない）</li>
<li>同一の売掛金を複数のファクタリング会社に売却（二重譲渡）</li>
<li>支払期日が極端に近い（7日以内）または遠い（120日超）</li>
</ul>
<h3>原因3：申込者に重大な問題がある</h3>
<p>以下の状況は、会社によっては対応困難：</p>
<ul>
<li>破産・民事再生など法的整理中</li>
<li>反社会的勢力への関与が疑われる</li>
<li>通帳に不審な入出金がある</li>
</ul>
<p><strong>なお、以下は多くの会社で審査通過できます：</strong></p>
<ul>
<li>赤字決算（売掛先が健全なら問題なし）</li>
<li>税金の滞納（条件によっては対応可能）</li>
<li>銀行融資の審査落ち（別の評価基準のため影響しない）</li>
</ul>
<hr>
<h2>審査が甘い・通りやすいおすすめファクタリング会社</h2>
<h3>1位：アクセルファクター（審査通過率93%以上）</h3>
<p>| 項目 | 内容 |
|------|------|
| 審査通過率 | <strong>93%以上</strong>（業界最高水準） |
| 手数料 | 0.5〜12% |
| 入金スピード | 最短2時間 |
| 赤字決算 | 対応可能 |
| 税金滞納 | 条件によって対応可能 |</p>
<p><strong>審査が通りやすい理由：</strong> 売掛先の信用力を最優先に評価するため、申込企業の財務状況が悪くても通過できるケースが多い。審査通過率93%以上は業界トップ水準。</p>
<h3>2位：ベストファクター（審査通過率92.25%）</h3>
<p>| 項目 | 内容 |
|------|------|
| 審査通過率 | <strong>92.25%</strong> |
| 手数料 | 2〜20% |
| 入金スピード | 最短即日 |
| 対応金額 | 10万円〜1億円 |
| 個人事業主 | OK |</p>
<p><strong>審査が通りやすい理由：</strong> 中小企業・個人事業主に特化した審査基準を持つ。10万円からの少額にも対応しており、審査完了まで平均5分。</p>
<h3>3位：メンターキャピタル（審査通過率90%超）</h3>
<p>| 項目 | 内容 |
|------|------|
| 審査通過率 | <strong>90%超</strong> |
| 手数料 | 2〜20% |
| 入金スピード | 最短即日 |
| 赤字・税金滞納 | <strong>対応可能</strong> |
| 特徴 | 厳しい状況でも柔軟対応 |</p>
<p><strong>審査が通りやすい理由：</strong> 赤字決算・税金滞納・銀行融資否決後でも積極的に審査。「他社で断られた」という状況でも対応できるケースが多い。</p>
<h3>4位：ラボル（開業直後・個人間取引OK）</h3>
<p>| 項目 | 内容 |
|------|------|
| 審査通過率 | 非公開（高水準） |
| 手数料 | 一律10% |
| 入金スピード | 最短30分 |
| 開業直後 | <strong>OK</strong> |
| 個人間取引 | <strong>OK</strong> |</p>
<p><strong>審査が通りやすい理由：</strong> 開業直後でも申込可能。個人間の取引（個人顧客への請求書）にも対応しており、通常のファクタリングでは難しいケースにも対応。</p>
<h3>5位：みんなのファクタリング（柔軟審査）</h3>
<p>| 項目 | 内容 |
|------|------|
| 手数料 | 3〜15% |
| 入金スピード | 最短2時間 |
| 対応金額 | 制限なし |
| 個人事業主 | OK |
| 特徴 | 柔軟な審査姿勢 |</p>
<p><strong>審査が通りやすい理由：</strong> 規模に関わらず柔軟な審査を行い、中小・零細企業や個人事業主に対応。</p>
<hr>
<h2>審査に通過するための準備</h2>
<h3>必要書類を揃える</h3>
<p>審査の精度を上げるために、以下の書類を準備しておきましょう：</p>
<p><strong>必須書類：</strong></p>
<ul>
<li>売掛金の請求書（申込する売掛金の証明）</li>
<li>直近6ヶ月の通帳コピー</li>
</ul>
<p><strong>あると審査が有利になる書類：</strong></p>
<ul>
<li>売掛先との基本契約書</li>
<li>発注書・納品書（取引実態の証明）</li>
<li>決算書（直近2期分）</li>
</ul>
<h3>申込時のポイント</h3>
<p><strong>1. 信用力の高い売掛先の請求書を選ぶ</strong>
複数の売掛先がある場合、より信用力の高い企業への請求書でファクタリングを申し込む。</p>
<p><strong>2. 売掛金の実在性を証明できる書類を準備する</strong>
請求書だけでなく、納品書・検収書・契約書を揃えることで、架空請求の疑いを払拭できます。</p>
<p><strong>3. 通帳の説明ができるようにしておく</strong>
大きな入出金がある場合、理由を説明できるようにしておくと審査がスムーズです。</p>
<hr>
<h2>まとめ</h2>
<p>ファクタリングの審査は、銀行融資と比べて「甘い」ことは事実です。しかし、それは「誰でも無条件で通る」という意味ではありません。</p>
<p><strong>審査通過のカギは「売掛先の信用力」：</strong></p>
<ul>
<li>上場企業・大手企業・官公庁への請求書があれば、自社の状況が悪くても通過できる可能性が高い</li>
<li>赤字・税金滞納・銀行融資落ちでも諦める必要はない</li>
</ul>
<p><strong>審査に不安な方には：</strong>
アクセルファクター（93%）、ベストファクター（92.25%）、メンターキャピタル（90%超）の3社が審査通過率の高いおすすめ会社です。</p>
<hr>
<h2>よくある質問（FAQ）</h2>
<p><strong>Q. 創業1年未満でもファクタリングは利用できますか？</strong></p>
<p>A. 会社によります。ラボルは開業直後でも利用可能です。一般的に創業間もない場合は、より厳しい審査になる傾向があります。</p>
<p><strong>Q. 消費者金融の借入が多くてもファクタリングは利用できますか？</strong></p>
<p>A. ファクタリングは借入ではなく売掛金の売却なので、消費者金融の借入状況は審査に直接影響しません。信用情報の照会をしない会社がほとんどです。</p>
<p><strong>Q. 決算書が赤字でも審査に通りますか？</strong></p>
<p>A. 通過できる可能性があります。メンターキャピタルやアクセルファクターは赤字決算でも対応しています。審査では自社の財務よりも売掛先の信用力が重視されます。</p>
<p><strong>Q. 審査に落ちる原因は何ですか？</strong></p>
<p>A. 主な原因は①売掛先の信用力が低い（個人・設立間もない会社等）、②売掛金の実在性に疑問がある、③申込者が破産・民事再生中、④二重譲渡の疑いがある、などです。</p>
<p><strong>Q. 複数のファクタリング会社に同時に申し込んでもいいですか？</strong></p>
<p>A. 同じ売掛金を複数社に申し込むことは「二重譲渡」になる可能性があり、避けてください。ただし、異なる売掛金で複数社に申し込むことは問題ありません。</p>

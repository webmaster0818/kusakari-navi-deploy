11:T29c6,<p>「節税を頑張りすぎて融資が通らなかった」――こんな失敗談を聞いたことはありませんか？</p>
<p>個人事業主にとって確定申告は税金を抑えるための重要な手続きですが、<strong>融資審査では確定申告書が「成績表」として見られる</strong>ことを忘れてはいけません。税金を減らそうとして所得を低く申告すると、いざ融資を受けたいときに「返済能力が低い」と判断され、審査に落ちるリスクがあります。</p>
<p>この記事では、融資審査で確定申告書のどこが見られるのか、赤字申告の影響、そして融資に強い申告のポイントを解説します。</p>
<hr>
<h2>融資審査で確定申告書のどこが見られる？</h2>
<p>金融機関が融資審査で確認するのは、主に以下の項目です。</p>
<h3>確定申告書B（第一表）のチェックポイント</h3>
<p>| チェック項目 | 記載箇所 | 審査での見方 |
|------------|---------|------------|
| 事業所得 | 営業等（ア） | 本業の稼ぐ力を確認 |
| 所得合計 | 所得金額の合計（⑫） | 返済能力の基準値 |
| 課税される所得金額 | （㉖） | 実質的な利益を確認 |
| 税額 | 所得税額 | 所得の妥当性を裏付け |</p>
<h3>青色申告決算書のチェックポイント</h3>
<p>| チェック項目 | 審査での見方 |
|------------|------------|
| 売上高 | 事業規模の確認、前年比の推移 |
| 売上原価 | 粗利率の確認 |
| 経費の内訳 | 不自然に高い経費がないか |
| 青色申告特別控除 | 65万円控除＝複式簿記＝管理能力が高い |
| 貸借対照表 | 資産・負債の状況 |</p>
<p><strong>特に重要なのは「売上高」と「所得金額」の2つです。</strong> 売上が順調に伸びていて、所得がしっかり残っている申告書が最も審査に通りやすくなります。</p>
<hr>
<h2>節税しすぎると融資が通らない問題</h2>
<h3>なぜ節税が融資のマイナスになるのか</h3>
<p>節税の基本は「経費を増やして所得を減らす」ことです。しかし融資審査では、<strong>所得＝返済能力</strong>と判断されます。</p>
<p><strong>具体例:</strong></p>
<p>| パターン | 売上 | 経費 | 所得 | 融資審査の印象 |
|---------|------|------|------|-------------|
| 節税重視 | 800万円 | 700万円 | 100万円 | 返済能力が低い |
| バランス型 | 800万円 | 500万円 | 300万円 | 返済余力がある |</p>
<p>パターンAの場合、「年間100万円の所得では毎月の返済が厳しい」と判断され、融資額が減額されるか、審査に落ちる可能性が高くなります。</p>
<h3>経費の水増しは絶対NG</h3>
<p>融資のために経費を減らすのではなく、<strong>そもそも経費の水増しをしない</strong>ことが大前提です。税務調査で経費の否認が発覚すると、修正申告が必要になり、融資の信頼性も一気に失われます。</p>
<hr>
<h2>赤字申告が融資に与える影響</h2>
<h3>赤字＝即アウトではない</h3>
<p>赤字申告だからといって、必ず融資に落ちるわけではありません。重要なのは<strong>赤字の原因と今後の見通し</strong>です。</p>
<p>| 赤字の種類 | 融資への影響 |
|-----------|-----------|
| 開業初年度の赤字 | 比較的寛容に見られる |
| 設備投資による一時的な赤字 | 理由の説明ができれば許容範囲 |
| 減価償却費による帳簿上の赤字 | 実質キャッシュフローがプラスなら問題なし |
| 売上低迷による慢性的な赤字 | 審査で厳しく見られる |
| 2期連続以上の赤字 | かなり厳しい |</p>
<p>特に注目すべきは「減価償却費」です。減価償却費は帳簿上の経費であり、<strong>実際にお金が出ていくわけではありません</strong>。したがって、「所得＋減価償却費」がプラスであれば、実質的なキャッシュフローは黒字と説明できます。</p>
<p>赤字でも融資を受けたい方は、<a href="/articles/article-akaji-yuushi/">赤字でも融資は受けられる？5つの方法</a>もあわせてご覧ください。</p>
<hr>
<h2>融資に強い確定申告のポイント5つ</h2>
<h3>ポイント1: 青色申告65万円控除を活用する</h3>
<p>白色申告ではなく、<strong>必ず青色申告（65万円控除）を選択</strong>しましょう。青色申告をしていること自体が「きちんと帳簿管理をしている」という証明になり、審査でプラスに働きます。</p>
<h3>ポイント2: 売上は正確に計上する</h3>
<p>売上の計上漏れや、意図的な売上の先送りは厳禁です。融資審査では通帳の入金記録と申告書の売上を照合されるため、不一致があると信用を失います。</p>
<h3>ポイント3: 経費は適正な範囲にとどめる</h3>
<p>経費を「使えるだけ使う」のではなく、<strong>事業に本当に必要な経費だけを計上する</strong>のが融資を見据えた申告のコツです。</p>
<p>特に注意すべき経費項目：</p>
<ul>
<li><strong>接待交際費:</strong> 売上に対して比率が高すぎると不自然</li>
<li><strong>旅費交通費:</strong> プライベートとの区分が曖昧になりやすい</li>
<li><strong>消耗品費:</strong> まとめ買いによる年末の駆け込み計上</li>
<li><strong>家事按分:</strong> 自宅兼事務所の按分比率が高すぎないか</li>
</ul>
<h3>ポイント4: 貸借対照表を整備する</h3>
<p>青色申告65万円控除を受けるために作成する貸借対照表は、融資審査でも重要な資料です。</p>
<ul>
<li><strong>事業主貸・事業主借が大きすぎないか：</strong> 生活費と事業費の区分が曖昧だと管理能力を疑われる</li>
<li><strong>売掛金の残高：</strong> 回収状況の健全性</li>
<li><strong>借入金の残高：</strong> 既存の借入状況</li>
</ul>
<h3>ポイント5: 3期分の申告書を揃える</h3>
<p>融資審査では通常<strong>直近2〜3期分の確定申告書を求められます</strong>。売上が右肩上がり、所得も安定している推移が理想的です。開業から2期以上経過してから融資を申し込むと、審査材料が揃うため通りやすくなります。</p>
<hr>
<h2>確定申告書以外に融資で見られる書類</h2>
<p>確定申告書だけでなく、以下の書類も審査対象です。</p>
<p>| 書類 | 確認されるポイント |
|------|-----------------|
| 通帳（6か月〜1年分） | 売上の入金パターン、資金の流れ |
| 借入の返済明細 | 既存の返済状況、延滞の有無 |
| 事業計画書 | 将来の売上見込み、成長性 |
| 見積書・契約書 | 受注の確実性 |
| 資金繰り表 | 返済計画の妥当性 |</p>
<p>特に<strong>通帳の記録は確定申告書の裏付け</strong>となるため、日頃から事業用口座とプライベート口座を分けて管理することが重要です。</p>
<hr>
<h2>融資を見据えた申告スケジュール</h2>
<p>| 時期 | やるべきこと |
|------|-----------|
| 4〜12月 | 毎月の帳簿を正確につける、経費は領収書を整理 |
| 1月 | 年間の収支を集計、所得金額を確認 |
| 2〜3月 | 確定申告書を提出（e-Taxで65万円控除） |
| 申告後 | 融資申込の準備（申告書の控え・通帳を整備） |</p>
<p><strong>融資を検討している年の確定申告は特に慎重に。</strong> 過度な節税をせず、返済能力を示せる所得水準を確保しましょう。</p>
<hr>
<h2>確定申告を税理士に依頼すべきか？</h2>
<p>融資を受ける予定がある場合、<strong>税理士に申告を依頼するメリットは大きい</strong>です。</p>
<ul>
<li>税理士の署名がある申告書は金融機関からの信頼度が高い</li>
<li>融資に通りやすい申告のアドバイスが受けられる</li>
<li>税務調査のリスクが低くなる</li>
<li>金融機関の紹介を受けられることもある</li>
</ul>
<p>費用は年間10〜30万円程度ですが、融資額が大きくなるほど税理士費用は「投資」として回収できます。</p>
<hr>
<h2>まとめ</h2>
<p>確定申告と融資は密接に関連しています。節税を重視しすぎると融資審査で不利になり、必要なときに資金を借りられないリスクがあります。</p>
<p><strong>融資に強い申告の鉄則:</strong></p>
<ol>
<li><strong>青色申告65万円控除を必ず活用する</strong></li>
<li><strong>経費は適正な範囲にとどめる（水増しNG）</strong></li>
<li><strong>売上は正確に・もれなく計上する</strong></li>
<li><strong>貸借対照表を整備する</strong></li>
<li><strong>3期分の申告書を揃えてから融資を申し込む</strong></li>
</ol>
